How to Create a Table in HTML

Elements of a Table

Order Element Description
1 table Add this to begin your table 00
2 00
3 thead Top row of the table. Include table row, tr, and then table data, td, as many times as you need columns. Be sure to include titles of each column between td. 00
4 tbody The main body of the table. Add tr as many times you have items and td as many times as you have columns. This pattern will repeat to get the length of the table. 00
5 tfooter This element is used at the bottom of the table. Use this to total columns or summarize your table. 00
6 colspan Used for data to span indicated number of columns. I used it here to display feature.
7 rowspan This will expand rows by number indicated. I used this element in the td first column to show you how to use it. 00
8 00

CSS Elements to Style Your Table

An external style sheet uses file name .css. An internal stylesheet is written using the style element inside the head element of an HTML file. CSS always placed between {} brackets with a style: and ; after each style. There are two ways to add style to your page.

  1. Embed CSS into HTML code
  2. Create CSS styles file

Makes to set html elements with a class and/or id to identify where your css style should be applied.
